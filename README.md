# 폴더 보기 (Folder Viewer)

현재 포커스된 문서가 속한 폴더의 파일 목록을 실시간으로 사이드바에 표시하는 Obsidian 플러그인입니다.

## 주요 기능

- **동적 파일 목록**: 현재 활성화된 파일이 속한 폴더의 모든 파일을 자동으로 표시
- **실시간 업데이트**: 파일을 전환할 때마다 자동으로 해당 폴더의 목록으로 업데이트
- **수정 시간 순 정렬**: 가장 최근에 수정된 파일이 목록 맨 위에 표시
- **활성 파일 하이라이트**: 현재 열린 파일을 시각적으로 강조 표시
- **빠른 파일 열기**: 목록에서 파일을 클릭하여 즉시 열기

## 사용 방법

### 뷰 열기

명령 팔레트(`Ctrl/Cmd + P`)를 열고 다음 중 하나를 선택합니다:

- **왼쪽 사이드바에 열기**: 왼쪽 사이드바에 폴더 파일 목록을 표시
- **오른쪽 사이드바에 열기**: 오른쪽 사이드바에 폴더 파일 목록을 표시

### 작동 방식

1. 플러그인 뷰를 열면 현재 활성화된 파일이 속한 폴더의 모든 파일이 표시됩니다
2. 다른 파일로 전환하면 해당 파일의 폴더 목록으로 자동 업데이트됩니다
3. 목록에서 파일을 클릭하면 해당 파일이 열립니다
4. 현재 열린 파일은 하이라이트되어 표시됩니다
